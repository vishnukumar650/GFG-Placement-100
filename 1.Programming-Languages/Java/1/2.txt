IDE

    IDEs increase programmer productivity by combining common activities of writing software into a single application: editing source code, building executables, and debugging.

First Java Program
    // Hello World Program in Java - Single Line Comment
    public class HelloWorld{
        /* This the main function,
           this is where the execution of
           a java program begins */
        public static void main(String args[]){
            System.out.println("Hello World!");
        }
    }


public: access specifier
class: keyword to create a class in java
HelloWorld: name of the class
static: access modifier
void: return type
main: method name
String args[]: String array named args for reading command line arguments to the main program.
System.out.println: https://www.geeksforgeeks.org/system-out-println-in-java/



Running the code:
    $ javac HelloWorld.java     // Compilation - .java (source code) to .class (byte code)
    $ java HelloWorld           // Running the byte code (JVM interprets the byte code)
